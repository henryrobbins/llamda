Common ideas in No.1 algorithm:
- Utilizing a random initial solution
- Refining the solution through local search
- Updating heuristics matrix based on solution improvement

{Design idea inspired by No.1 algorithm for No.2 algorithm: 
Based on the updated heuristics matrix, prioritize nodes with higher indicators to select the next node with the shortest distance for a more informed sampling strategy to find a better solution.}

```python
import numpy as np

def heuristics(distance_matrix):
    n = len(distance_matrix)
    heuristics_matrix = np.zeros((n, n))
    
    for _ in range(n):
        available_nodes = list(range(n))
        current_node = np.random.choice(available_nodes)
        available_nodes.remove(current_node)
        
        while available_nodes:
            next_node = max(available_nodes, key=lambda x: heuristics_matrix[current_node][x] * (-1))
            heuristics_matrix[current_node][next_node] += 1
            heuristics_matrix[next_node][current_node] += 1
            available_nodes.remove(next_node)
            current_node = next_node
    
    return heuristics_matrix
```
