{
The design idea of the new algorithm is to incorporate a dynamic factor that adjusts the selection of the next node based on a combination of the current shortest distance and the frequency of edges already chosen, leading to a more diverse exploration of potential solutions.
}

```python
import numpy as np

def heuristics(distance_matrix):
    n = len(distance_matrix)
    heuristics_matrix = np.zeros((n, n))
    
    for _ in range(n):
        available_nodes = list(range(n))
        current_node = np.random.choice(available_nodes)
        available_nodes.remove(current_node)
        
        while available_nodes:
            next_node = min(available_nodes, key=lambda x: distance_matrix[current_node][x] / (1 + heuristics_matrix[current_node][x]))
            heuristics_matrix[current_node][next_node] += 1
            heuristics_matrix[next_node][current_node] += 1
            available_nodes.remove(next_node)
            current_node = next_node
    
    return heuristics_matrix
```
