{
The new algorithm calculates the heuristics matrix by adding a small epsilon value to the reciprocal of the distances in the distance matrix.
}

```python
import numpy as np

def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    heuristics_matrix = np.zeros((num_nodes, num_nodes))
    epsilon = 1e-6

    for i in range(num_nodes):
        for j in range(num_nodes):
            if i != j:
                heuristics_matrix[i, j] = 1 / (distance_matrix[i, j] + epsilon)

    return heuristics_matrix
```
