{
The new algorithm calculates the heuristics matrix by taking the squared reciprocal of the distances in the distance matrix and then subtracting the minimum value in the resulting matrix.
}

```python
import numpy as np

def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    heuristics_matrix = np.zeros((num_nodes, num_nodes))

    for i in range(num_nodes):
        for j in range(num_nodes):
            if i != j:
                heuristics_matrix[i, j] = 1 / (distance_matrix[i, j] ** 2)

    min_val = np.min(heuristics_matrix)
    heuristics_matrix -= min_val

    return heuristics_matrix
```
