The new algorithm uses a different score function that takes the reciprocal of the square root of the distances in the distance matrix.
}

Code:
```python
import numpy as np

def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    heuristics_matrix = np.zeros((num_nodes, num_nodes))

    for i in range(num_nodes):
        for j in range(num_nodes):
            if i != j:
                heuristics_matrix[i, j] = 1 / np.sqrt(distance_matrix[i, j])

    return heuristics_matrix
```
