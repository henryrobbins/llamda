{
The new algorithm calculates the heuristics matrix by summing the distances in the distance matrix and then dividing each element in the heuristics matrix by the corresponding sum.
}

```python
import numpy as np

def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    heuristics_matrix = np.zeros((num_nodes, num_nodes))
    
    row_sum = np.sum(distance_matrix, axis=1)
    col_sum = np.sum(distance_matrix, axis=0)
    
    for i in range(num_nodes):
        for j in range(num_nodes):
            if i != j:
                heuristics_matrix[i, j] = distance_matrix[i, j] / (row_sum[i] + col_sum[j])
    
    return heuristics_matrix
```
