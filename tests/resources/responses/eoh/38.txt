{
The new algorithm calculates the heuristics matrix by taking the product of the distances and their inverse in the distance matrix and then subtracting the maximum value in the resulting matrix.
}

```python
import numpy as np

def heuristics(distance_matrix):
    num_nodes = distance_matrix.shape[0]
    heuristics_matrix = np.zeros((num_nodes, num_nodes))
    
    for i in range(num_nodes):
        for j in range(num_nodes):
            if i != j:
                heuristics_matrix[i, j] = distance_matrix[i, j] * (1 / distance_matrix[i, j])
                
    max_val = np.max(heuristics_matrix)
    heuristics_matrix -= max_val

    return heuristics_matrix
```
